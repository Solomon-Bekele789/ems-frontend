# EMS Frontend 🖥️

React frontend for the Employee Management System.

## Tech Stack
- React + Vite
- Tailwind CSS
- React Router
- Axios

## Features
- Login with JWT authentication
- Dashboard with live stats
- Employee CRUD
- Department management
- Attendance tracking
- Leave management

## Backend
🔗 [EMS Backend](https://github.com/Solomon-Bekele789/ems-backend)

## Run Locally
npm install
npm run dev
